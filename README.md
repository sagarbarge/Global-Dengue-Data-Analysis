# Global-Dengue-Data-Analysis

Global Dengue Data Analysis (Power BI Dashboard)

Overview
This interactive Power BI dashboard visualizes global dengue virus trends, transmission patterns, and case distributions to support public health monitoring and resource allocation.

Data Source
Source: World Health Organization (WHO) Data Portal

URL: https://data.who.int/

Key Features
Global Mapping: Visualizes dengue hot spots and case distribution by WHO regions.

Trend Analysis: Tracks seasonal spikes, monthly case counts, and Year-over-Year (YoY) changes.

Country Drill-Down: Allows localized analysis and side-by-side regional comparisons.

Tech Stack
ETL: Power Query for data cleansing and normalization.

Modeling: Star schema design mapping epidemiological data to Geography and Calendar dimensions.

Analytics: Custom DAX measures for dynamic time-intelligence and case aggregations.
